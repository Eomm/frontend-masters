# Notes

## Guiding principles

- Grid based layout
- Images that resize
- Media queries

## Flaxbox

- Flex containers are the parent elements
- Flex items are the children elements
- Main axis is the direction of the flex items (horizontal or vertical)
- Disadvantage: Works in 1 dimension
